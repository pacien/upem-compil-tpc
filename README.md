# UPEM / Compilation / Projet

Compilateur TPC (Très Petit C).


## Compilation

* Compiler le compilateur : `make all`
* Générer le rapport PDF : `make out/rapport.pdf` (requiert `pandoc`)


## Dépendances

* Analyseur lexical : flex
* Analyseur syntaxique : bison
